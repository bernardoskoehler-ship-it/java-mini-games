# Java Mini Games

A collection of small Java games created to practice programming logic, object-oriented programming, and problem-solving.

## About

This repository contains mini game projects developed during my Java learning journey.

The goal is to improve:

* Java fundamentals
* OOP concepts
* code organization
* problem-solving skills
* project structure

Each game is built as a practical way to apply what I study and track my progress over time.

## Projects

### v1 Tic-Tac-Toe

A console-based Tic-Tac-Toe game.

Features:

* two-player mode
* board display in the console
* position validation
* occupied space validation
* win condition check (rows, columns, diagonals)

## Technologies Used

* Java
* Object-Oriented Programming
* Scanner (user input)

This repository was created for study purposes and to document my progress while learning Java.

It will continue to grow with new mini games and improved versions of existing projects.
