# 🗺️ Grid-Based Console Game (Java)

A simple grid-based console game built in Java, focused on movement, interaction, and object-oriented design.

## 🎮 About the Project

This project simulates a character moving inside a 2D map, interacting with objects such as keys, doors, and obstacles.

The main goal is to practice **Object-Oriented Programming (OOP)** concepts while building a small interactive system.

## ⚙️ Features

- Character movement (W, A, S, D)
- 2D grid map (7x7)
- Item interaction system
- Inventory using `Map`
- Door unlocking with key
- Obstacles (walls)
- Console rendering

## 🧱 Technologies

- Java
- Object-Oriented Programming (OOP)
- HashMap (Inventory system)

## 🧠 Concepts Practiced

- Classes and Objects
- Inheritance
- Abstraction
- Encapsulation
- Composition
- Data structures (`Map`)

## 🕹️ Controls

| Key | Action       |
|-----|-------------|
| W   | Move up     |
| S   | Move down   |
| A   | Move left   |
| D   | Move right  |
| E   | Interact    |
